# Vi-SaaS: AI-Powered Visa Prediction Platform 🛂

**Vi-SaaS** is a comprehensive AI/ML solution designed to predict visa approval probabilities and estimate processing times. By combining advanced Machine Learning models with the power of Google Gemini 2.5 Flash, the platform provides users with personalized insights, expert guidance, and actionable risk factor analysis to maximize their application success.

---

## 🚀 Key Features

- **Random Forest AI**: An ensemble model trained on 12,000+ records, delivering high-accuracy approval predictions.
- **Processing Time Estimation**: Linear Regression models that predict wait times across 30+ destination countries and 7+ visa types.
- **Gemini 2.5 Flash Insights**: Personalized expert recommendations and document checklists generated by Google's latest LLM.
- **Interactive Dashboard**: A sleek, dark-themed UI for managing predictions and tracking application history.
- **Global Coverage**: Risk scoring and specific requirements for 50+ nationalities and major global destinations (Schengen, US, UK, Canada, Australia, etc.).

---

## 📂 Project Milestone Structure

This repository is organized into four main milestones that represent the end-to-end Machine Learning pipeline:

### [Milestone 1: Dataset Preprocessing & Cleaning](./Milestone_1_Dataset)
- **Objective**: Lay the foundation by acquiring and cleaning the operational dataset.
- **Contents**:
  - `visa_dataset.csv`: The primary dataset containing 12,000+ synthetic yet realistic records.
  - `milestone_1_AI_visa_prediction_dataset.py`: Python script for data loading, missing value handling, and initial feature identification.

### [Milestone 2: Exploratory Data Analysis (EDA)](./Milestone_2_EDA_Visualization)
- **Objective**: Gain deep insights into factors influencing visa wait times and approval rates.
- **Contents**:
  - `milestone_2_eda_feature_engineering.py`: Advanced visualization script using Seaborn and Matplotlib to analyze correlations between nationality, income, and processing cycles.

### [Milestone 3: Predictive Modeling & Training](./Milestone_3_Model_Training)
- **Objective**: Build and evaluate the core predictive engines of the platform.
- **Contents**:
  - `milestone_3_regression_modelling.py`: Benchmarking and fine-tuning regression models for processing time estimation.
  - `train_models.py`: The production-ready training script that generates the serialized models used by the API.

### [Milestone 4: The Final Vi-SaaS System](./Milestone_4_Vi-SaaS_System)
- **Objective**: Deploy the integrated, full-stack AI application.
- **Contents**:
  - `backend/`: FastAPI-powered server handling authentication, ML inference, and Gemini integration.
  - `frontend/`: Modern, responsive web interface built with HTML5, CSS3, and Vanilla JS.
  - `start_backend.bat` & `start_frontend.bat`: One-click startup scripts for local execution.

## 🛠️ Step-by-Step Setup Guide

Follow these instructions to clone and run the system successfully on your local machine.

### 1. Clone the Repository
Open your terminal and run:
```bash
git clone https://github.com/your-username/VISAAS.git
cd VISAAS
```

### 2. Prerequisites
- **Python 3.9+** must be installed on your system.
- Ensure Python is added to your environment `PATH`.
- (Optional) A Google Gemini API Key if you wish to use the advanced AI insights feature.

### 3. Launch the Backend (API Server)
1. Navigate to the system backend folder:
   ```bash
   cd "Milestone_4_Vi-SaaS_System/backend"
   ```
2. Run the startup script:
   ```bash
   start_backend.bat
   ```
   *This script will automatically install dependencies from `requirements.txt`, train the ML models (saving them to the `models/` folder), and launch the FastAPI server on `http://localhost:5000`.*

### 4. Launch the Frontend (UI)
1. Open a **new** terminal window and navigate to the Milestone 4 folder:
   ```bash
   cd "Milestone_4_Vi-SaaS_System"
   ```
2. Run the frontend startup script:
   ```bash
   start_frontend.bat
   ```
   *This starts a local development server on `http://localhost:3000`.*

### 5. Access the Platform
- Open your browser and visit: **[http://localhost:3000](http://localhost:3000)**
- Explore the API documentation at: [http://localhost:5000/docs](http://localhost:5000/docs)
